# porfolio-2.0
Hey hope you are doing well. Here I will help you to setup my project in you computer.

Step 1: First create a folder on your machine where you want to keep my code. Now open it in your favorite code editor mine one is VS Code. Once you open the folder in your code editor open its terminal (just click on view in top left and there you can see the terminal word, just click on it)

Step 2: Now come to my git hub repo (https://github.com/ut-pro/porfolio-2.0)

Step 3: When you come to the repo you will see a green button on top right of the page which says "<> Code", click on it.

Step 4: When you click it you can see HTTPS link (https://github.com/ut-pro/porfolio-2.0.git), just copy it.

Step 5: Now come back to the code editor in the terminal type
- git clone <repo link> or git clone https://github.com/ut-pro/porfolio-2.0.git
then press enter.

Step 6: A new folder has is created and all the code from GitHub is now inside your folder
